# Breast Cancer Classification using CNN
# Overview
This project builds a deep learning model to classify breast cancer histopathology images as benign or malignant. A Convolutional Neural Network (CNN) is trained on the IDC (Invasive Ductal Carcinoma) dataset to assist in early detection of breast cancer.

# Technologies Used
Python

TensorFlow / Keras

NumPy

Matplotlib

Jupyter Notebook

# Dataset
The model uses the IDC Breast Cancer Histology Image Dataset, which contains labeled images of breast tissue:

0 – Benign
1 – Malignant

# Dataset Source:
https://www.kaggle.com/datasets/paultimothymooney/breast-histopathology-images
